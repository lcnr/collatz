Form a sequence by beginning with an integer n and repeatedly applying the function f(n), taking the result of each step as the input of the next.

f(n) := n / 2       if (n % 2 == 0)
        3n + 1      otherwise

The collatz conjecture states that for any natural number n this sequence will reach the loop 1, 4, 2, 1.

This project

My goal is to prove that no other loop is possible, meaning that this sequence must either diverge to infinity or reach 1. While experimenting with different ideas was enjoyable, this goal has not been reached.
